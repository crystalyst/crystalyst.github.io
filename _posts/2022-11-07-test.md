#### Set up an IDE 

VSCode is a free-of-charge integrated development environment (IDE) for developers. As it provides powerful extensions and consumes only a fraction of RAM while running, I decided to use VSCode for my project moving forward. Few setups I did were listed below:

- Open VSCode from my local terminal directly [Referred Document Link](https://www.freecodecamp.org/news/how-to-open-visual-studio-code-from-your-terminal/)
- Configure two extensions for code formatting, ESLint and Prettier

#### Initialize Nodejs project

Node.js allows users to build a server locally using JavaScript, like other programming languages, such as Python or Java (Originally, JavaScript was only used on Web Browser Console for implementing interactive actions on frontend). Few setups I did were listed below:

- Install Node.js using pkg installer [NodeJS Downloader](https://nodejs.org/en/download/)
- Initialize a Nodejs project named "textbook" using npm (Node Package Manager)

#### Configure and publish Jekyll blog

Among many options for personal blogging, I chose Github pages as it provides a free hosting service from Github Repositories directly. In terms of themes, I chose [contrast Jekyll theme](https://github.com/niklasbuschmann/contrast) to begin with.
